# MED-Workshop
Materials for the MED Workshop as part of the DAVoS (Digital Analysis and Visualisation of Simulations) series at the University of Birmingham, lead by Darius Khoshdel.

You can click on the [`MED-Workshop.ipynb`](https://github.com/uob-positron-imaging-centre/MED-Workshop/blob/main/MED-Workshop.ipynb) file to see a pre-rendered version of the main notebook; it contains all instructions to run the code cells.
